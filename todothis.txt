Problem 1: Pending Pets
Images are in D:\Petpostedimages\ but frontend tries to load from regular /images/ URL.
Fix: Use /images/pending/ URL instead!
Problem 2: Adoption Requests
The adoption_requests table probably doesn't have the pet's image stored in the pet_image column. It's NULL or empty!
Fix: Copy image from pets table to adoption_requests table.

JOIN pets p ON ar.pet_id = p.pet_id
SET ar.pet_image = p.image_url
